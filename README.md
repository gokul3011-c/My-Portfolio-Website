# My Portfolio Website

Static personal portfolio for Gokul Sowri Galla.

## Local preview

Open `index.html` in a browser, or run a local server:

```bash
npx serve .
```

## Deploy

This is a static site. Deploy the repository root on:

- **Vercel:** Import the repo and deploy with default settings (no build command).
- **Netlify:** Import the repo; publish directory is `.` (already set in `netlify.toml`).
- **GitHub Pages:** Enable Pages from the `main` branch, folder `/ (root)`.
